JunkLoad

Current and future’s massive geometric 3D models exceed the capacity of common high-end GPUs for real-time rendering. Parallel rendering can be used as a general tool for the interactive visualization of massive polygonal 3D models, but standard mesh representations do not allow their easy adaption to efficient distributed cluster-parallel rendering.

In this project a parallel mesh visualization application is developed that can display massive meshes efficiently at interactive frame rates, reading mesh data at runtime from disk on demand (rendering from out-of-core) and distributing the rendering task to a number of parallel rendering nodes.

libMaoni

LibMaoni is a 3D rendering framework developed at the Visualization and Multimedia Lab (VMML) at the Insitute of Informatics of University of Zurich UZH. The goal is to implement a common 3D viewer platform which can be used as framework for the researchers’ graphics and visualization projects.

libMaoni was developed by Daniel Pfeifer and Stefan Hutter as a Master Project.
